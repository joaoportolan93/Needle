## Reporting a Vulnerability

We take the security of our software seriously. If you believe you have found a security vulnerability in Sonora, please report it to us as described below.

**Please do not report security vulnerabilities through public GitHub issues.**

### How to Report

To report a vulnerability, please email [INSERT SECURITY EMAIL OR CONTACT]. We will acknowledge receipt of your vulnerability report and strive to send you regular updates about our progress. We will handle all reports with strict confidentiality.

### Response Timeline

We are committed to addressing security vulnerabilities promptly. We will investigate all credible reports and, if a problem is confirmed, we will release a patch as soon as possible.
